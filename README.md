
##一个基于Tinymce上扩展的图片热点编辑器

###它主要有以下特点

1.自定义复杂热点(精确定位并,具有超链接,倒计时,锚点,等动作)    
2.支持定时切换图片    
3.支持当前图片上的文字进行样式设定    
4.支持自动将map转换为a
